#### 安装教程

1. clone
2. run npm install to install  `package.json`
3. 运行 `gulp`

#### 使用说明

1.根据 `gulpfile.js` 中的 `自定义文件夹路径` 新建对应的项目名，然后新建对应的二级文件夹，`_path` 代表源文件文件夹， `_outputOnce` 代表一级编译文件夹， `_outputPath` 代表最终编译文件夹。

3. 提交代码
4. 新建 Pull Request
